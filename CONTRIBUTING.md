# Contributing

Thanks for your interest in this Health & Medical Open-Source Hub!

## How to suggest a project

1. Open an issue with the repository URL and a short reason why it belongs here.
2. Projects should be genuinely health / medical / clinical / healthcare-AI focused.
3. Active, well-starred, open-source projects are preferred.

## Note

This catalog is regenerated automatically every week. Manual edits to the project list
may be overwritten — submit changes to the curation criteria instead via an issue.
