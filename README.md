## 🧰 The app/ project (runnable)

A small, tested Python toolkit for health data: BMI, blood pressure, heart-rate zones, and FHIR patient parsing.

```bash
cd app
pip install -r requirements.txt
pytest -q
python -m health_toolkit.cli bmi --weight 70 --height 1.75
```
